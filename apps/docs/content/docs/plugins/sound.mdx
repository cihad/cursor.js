## SoundPlugin

The `SoundPlugin` plays sound completely synchronized with the cursor's `click()` actions, giving user the audible feedback.

### Options

  path="../../../../../packages/core/src/plugins/SoundPlugin.ts"
  name="SoundPluginOptions"
/>

```ts
import { Cursor, SoundPlugin } from 'cursor.js';

const cursor = new Cursor();

// Install the plugin
cursor.use(new SoundPlugin());

// The click action will now trigger a sound
cursor.click('.submit-button');
```

You can optionally pass an `audioUrl` option to customize the click sound:

```ts
cursor.use(
  new SoundPlugin({
    audioUrl: '/my-custom-click-sound.mp3',
    volume: 0.8,
  }),
);
```
