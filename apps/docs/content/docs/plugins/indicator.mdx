## IndicatorPlugin

The `IndicatorPlugin` allows you to show contextual information by placing a floating element alongside the cursor.

### Demo

<iframe src="/demos/indicator" className="w-full h-80 rounded-xl border mt-6 overflow-hidden relative" />

### Options

| Option   | Type     | Default   | Description                                          |
|----------|----------|-----------|------------------------------------------------------|
| `color`  | `string` | `#ff4757` | The background color of the indicator element.       |
| `size`   | `number` | `40`      | The width size of the indicator line in pixels.      |

### Usage

```ts
import { Cursor, IndicatorPlugin } from 'cursor.js';

const cursor = new Cursor();
const indicator = new IndicatorPlugin();

cursor.use(indicator);

// Update indicator state whenever needed
indicator.update({
  text: 'Waiting for network...',
  color: 'blue'
});

// Remove or hide the indicator
indicator.hide();
```
