## `FloatingWaitForUserPlugin`

`FloatingWaitForUserPlugin` is the Floating UI powered variant of `WaitForUserPlugin`. It keeps the same `.waitForUser()` API, but anchors the handoff panel near the cursor with collision handling, flipping, shifting, and automatic repositioning.

## Installation

The plugin is included in the `@cursor.js/pro` package.

```bash
npm install @cursor.js/pro @cursor.js/core
```

## Basic Usage

```typescript
import { Cursor } from '@cursor.js/core';
import { FloatingWaitForUserPlugin, SpotlightPlugin } from '@cursor.js/pro';

const cursor = new Cursor();
cursor.use(new SpotlightPlugin());
cursor.use(new FloatingWaitForUserPlugin());

await cursor.waitForUser({
  target: '#confirm-delete',
  event: 'click',
  message: 'Your turn: confirm the deletion to continue.',
  spotlight: true,
  backdrop: true,
  speak: true,
});
```

## Without `FloatingWaitForUserPlugin`

If you do not want to use the dedicated plugin class, you can attach the same positioning behavior to the base `WaitForUserPlugin` manually:

```typescript
import { Cursor } from '@cursor.js/core';
import {
  SpotlightPlugin,
  WaitForUserPlugin,
  createFloatingWaitForUserPositioner,
} from '@cursor.js/pro';

const cursor = new Cursor();
cursor.use(new SpotlightPlugin());
cursor.use(
  new WaitForUserPlugin({
    positioner: createFloatingWaitForUserPositioner(),
  }),
);
```

## Speech and TTS

Like `SayPlugin` and `PromptPlugin`, this plugin emits `speech_requested` when a `message` is present. That means `SpeechPlugin` or `GeminiTTSPlugin` can narrate the handoff panel automatically.
